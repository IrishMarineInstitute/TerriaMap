About the Marine Atlas

The Atlas is a website for map-based access to spatial reference data from Irish government agencies. It is an initiative of the Marine Institute.

The Atlas is a fully open architecture. When you access data through it, you are typically accessing the data directly from the government department or agency which is the custodian of that data.

The Atlas...

  • provides easy access to authoritative and other spatial data to government, business and the public
  • facilitates the opening of data by national and local government bodies
  • provides an open framework of geospatial data services that supports commercial and community innovation

To see what data is available on the Atlas, refer to the Data Catalogue in the Atlas itself. Click the Data Catalogue tab and expand the Data Set group names.

How to use the Atlas

Just go to https://atlas.marine.ie/. Most operations are obvious, but you can click the Help and FAQs button near the top of this page for specific instructions.

For further information email datarequests@marine.ie. Feel free to send us feedback and bug reports.

Contributing Data to Atlas

Spatial data sets which are placed on data.marine.ie using supported data formats and protocols (such as WMS) will automatically be included in the Atlas. See the How To... page for more information.

Irish government agencies that wish to find out more information about contributing data sets to the Atlas should email datarequests@marine.ie with their questions. Typically, the Atlas does not store the data itself, but refers to the data servers of the data contributors, directly taking that data and presenting it overlaid over a map. Therefore, agencies providing data directly must have the data available via an online server using a standard protocol.

Atlas Software

The Atlas is Open Source software available as a Github project. The core platform, TerriaJS, was initially developed by Data61, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.
